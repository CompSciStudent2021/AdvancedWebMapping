# Gym Locator Application
The Gym Locator Application is an interactive web-based tool that helps users find nearby gyms based on their type, such as exercise stations or trim trails. Leveraging the power of OpenStreetMap and Leaflet, the application provides an intuitive map interface where users can:<br>

View Nearby Gyms: Discover various gyms in their vicinity marked on an interactive map.<br>
Filter Gyms by Type: Utilize a dropdown menu to filter gyms according to different types, making it easier to find specific locations.<br>
Get Directions: Obtain driving directions to selected gyms directly through Google Maps with a single click.<br>
User Location: The application detects the user's location and displays it on the map for better navigation.<br>
This project is built using Django, integrating Geographic Information Systems (GIS) capabilities to provide a seamless user experience for fitness enthusiasts seeking gym facilities nearby.
